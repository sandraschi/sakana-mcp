## Research Task Library

Curated sample tasks for fast onboarding and repeatable autonomous research loops.

### Layout

- `tasks/*.json`: task cards with objectives, constraints, success criteria
- Can be extended with custom domain-specific tasks

### Task Schema (minimal)

- `task_id`
- `title`
- `domain`
- `difficulty`
- `summary`
- `success_criteria` (array)
- `starter_prompt`
